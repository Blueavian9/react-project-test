# Contributing to HolisticBook

Thanks for considering contributing!

## How I Work (solo dev notes)
- Work in small PRs (one EPIC/PR from PRD.md at a time)
- Follow TASKS.md checklist
- Manual lint: `npx eslint . --fix`
- No pre-commit hooks
- Commit messages: conventional style (feat:, fix:, chore:, docs:)
- Always test locally with `wrangler dev`

## Pull Request Checklist
- [ ] Update TASKS.md checkboxes
- [ ] Update memory-bank.md
- [ ] All new code follows SOLID principles
- [ ] PHI fields encrypted where applicable
